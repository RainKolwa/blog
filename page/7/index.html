iScroll点击触发多次解决办法

发表于 2014-12-02 | 分类于 Magento

在点击事件中加入如下判断

方法1：iscroll初始化的时候加上这段配置

click:((/Chrome\/\d/.test(window.navigator.appVersion)) ? true : parseFloat(navigator.userAgent.match(/Android [\d+.]{3,5}/)[0].replace('Android ','')) < 4.4 ? false : true)
方法2： 
var t1 = null;//设置为全局变量
if (t1 == null){
    t1 = new Date().getTime();
}else{        
    var t2 = new Date().getTime();
    if(t2 - t1 < 500){
        t1 = t2;
        return;
    }else{
        t1 = t2;
    }
}

indexOf()与lastIndexOf()

发表于 2014-11-25 | 分类于 Magento

indexof方法返回子字符串第一次出现的位置，
相反，lastIndexOf方法返回子字符串最后一次出现的位置。
比如：

var str = 'I love javascript';
console.log(str.lastIndexOf('a'));//10
console.log(str.indexOf('a'));//8
